# 产品评论情感分析系统

## 项目简介
这是一个基于深度学习的中文产品评论情感分析系统，可以分析用户对产品/服务的评论文本，自动识别情感倾向，并生成可视化分析报告。

## 功能特点

### 核心功能
1. **中文评论文本情感分类**
   - 支持正面、负面、中性情感分类
   - 基于jieba分词和SnowNLP算法
   - 结合关键词匹配提高准确性

2. **用户情感维度精确提取**
   - 精确识别用户情感倾向
   - 计算情感得分（0-1之间）
   - 提取情感关键词

3. **多维度数据可视化**
   - 饼图展示情感分布
   - 柱状图展示评论数量统计
   - 情感得分分布图
   - 详细的评论列表展示

4. **批量处理支持**
   - 支持单个文件上传
   - 支持批量文件上传
   - 支持手动输入评论文本
   - 实时分析结果显示

### 支持格式
- TXT 文本文件
- CSV 文件
- XLSX Excel文件

## 安装和使用

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 运行系统
```bash
python app.py
```

### 3. 访问系统
打开浏览器访问：http://localhost:5000

## 使用说明

### 文件上传
1. 点击"上传评论文件"区域选择文件
2. 支持拖拽文件到指定区域
3. 点击"上传分析"按钮开始分析

### 手动输入
1. 在文本框中输入评论内容
2. 点击"分析单条评论"按钮
3. 查看分析结果

### 批量上传
1. 按住Ctrl键选择多个文件
2. 一次性上传多个文件
3. 系统会合并分析结果

- **后端**: Python Flask

- **数据处理**: pandas, numpy

## 系统特色

1. **中文优化**: 针对中文评论进行优化，包含大量中文情感词库
2. **实时分析**: 支持实时分析，即时展示结果
3. **可视化丰富**: 多种图表展示，直观易懂
4. **易用性**: 界面简洁，操作简单，支持拖拽上传
5. **扩展性好**: 支持多种文件格式，易于扩展新的分析功能

## 注意事项

1. 单个文件大小限制为16MB
2. 支持.txt、.csv、.xlsx格式文件
3. 建议文件编码为UTF-8
4. 评论文本应为中文
5. 确保文本内容完整，避免大量空行
